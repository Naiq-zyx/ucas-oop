# Assignment Description

This assignment contains two UML class diagrams, designed using **PlantUML**:

- `library.puml`: Library Management System, showing the design of Student, Book, and Librarian classes for borrowing and returning books.
- `car.puml`: Car Management System, designing car classes from the perspectives of manufacturer, owner, and traffic authority.

# 作业说明

本次作业包含两个 UML 类图，使用 PlantUML 设计：

- `library.puml`：图书馆管理系统，展示学生、书籍和管理员的类设计及借书还书场景。
- `car.puml`：汽车管理系统，分别从厂家、车主和交通管理机构三个视角设计汽车类。
